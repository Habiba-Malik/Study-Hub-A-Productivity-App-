# Study Hub (A Productivity App)

## Overview

Study Hub is a productivity application built using Object-Oriented Programming (OOP) concepts. It helps users note down and manage tasks, remove tasks when completed, and display them on an interactive calendar. This allows users to plan future activities and clearly see which tasks are scheduled for each day. The app also includes a dedicated study session section where users can listen to calming music and focus better on their studies.

---

## Features

* ✅ **Task Management**

  Add, remove, and manage daily tasks
* 📅 **Interactive Calendar**

  View and plan tasks across different dates
* 🎧 **Study Session Mode**

  Focus-enhancing environment with calming music
* 🧩 **OOP-Based Design**

  Structured using classes and modular programming principles

---

## Objectives

* Create an interactive task management system
* Develop a task-based calendar for planning and visualization
* Promote deep-focused learning and help reduce procrastination
